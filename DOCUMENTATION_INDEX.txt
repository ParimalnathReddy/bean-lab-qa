📊 OUTPUT DOCUMENTATION
================================================================================

What the processor creates:

1. data/processed_chunks.json
   └─ Main output: 15,000-20,000 chunks
   └─ Format: [{text, source_file, year_range, chunk_id, page, tokens}, ...]

2. data/processing_metadata.json
   └─ Statistics and configuration
   └─ Success/failure counts
   └─ Processing timestamp

3. data/checkpoint_N.json
   └─ Intermediate saves (every 50 PDFs)
   └─ For recovery if crash occurs

4. logs/processing.log
   └─ Detailed processing log
   └─ INFO/WARNING/ERROR messages

5. logs/pdf_processing_JOBID.{out,err}
   └─ SLURM job output
   └─ Standard output and errors

================================================================================
🔧 CUSTOMIZATION GUIDE
================================================================================

Want to change chunk size?
└─ Edit: --chunk-size parameter (default: 512)
└─ Effect: Larger = fewer chunks, more context
           Smaller = more chunks, more precise

Want to change overlap?
└─ Edit: --chunk-overlap parameter (default: 50)
└─ Effect: Larger = more context continuity
           Smaller = more independent chunks

Want to enable OCR?
└─ Add: --use-ocr flag
└─ Warning: 10x slower!

Want to process subset?
└─ Edit: --input-dir to point to subset directory

Want more frequent checkpoints?
└─ Edit: --batch-size parameter (default: 50)
└─ Effect: More checkpoints = safer but slower

================================================================================
🐛 COMMON ISSUES & SOLUTIONS
================================================================================

Issue: "No PDFs found"
├─ Check: data/pdfs/ directory exists
├─ Check: PDFs are linked or copied
└─ Solution: Run scripts/analyze_and_copy_pdfs.sh

Issue: "ImportError: No module named 'PyPDF2'"
├─ Check: Conda environment activated
└─ Solution: pip install -r requirements.txt

Issue: "Permission denied"
├─ Check: File permissions (ls -la)
└─ Solution: chmod +x scripts/*.sh

Issue: "Out of memory"
├─ Check: SLURM memory allocation
└─ Solution: Increase --mem in jobs/process_pdfs.sb

Issue: "Job timeout"
├─ Check: Time limit in SLURM script
└─ Solution: Increase --time in jobs/process_pdfs.sb
